# AB UI Tools (Scripts)

CLI tool for scaffolding new applications.

## Usage

```bash
npx @ab.uitools/scripts --name my-app --template react
```

## Templates

This CLI includes a **React application template** to generate a new app pre-wired with the AB UI Tools packages.  
Note: the template is **not ready yet** and will be added later.
